# cargoの基本コマンド
[引用元](https://qiita.com/udayan28/items/e59afd39a7ab16911c25)
- プロジェクト作成　：cargo new プロジェクト名
- ビルド　　　　　　：cargo build
- ビルド＆実行　　　：cargo run
- コンパイルチェック：cargo check(cargo c)
- リリースビルド　　：cargo build --release
